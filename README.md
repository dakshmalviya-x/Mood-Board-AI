## 📖 About The Project

MoodBoard AI is a full-stack journaling platform designed to make self-reflection more engaging, visual, and insightful.

Instead of simply storing journal entries, MoodBoard AI helps users understand their emotions through AI-powered mood analysis, visual mood tracking, and personalized reflection summaries.

The project combines modern UI design, smooth animations, and intelligent emotional insights to create a calming and premium user experience.

---

## ✨ Features

### 📝 Smart Journaling

* Create and manage personal journal entries
* Clean and distraction-free writing experience
* Organized mood history and archives

### 🤖 AI-Powered Mood Analysis

### 🎨 Mood Visualization

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/your-username/moodboard-ai.git

cd moodboard-ai
```

---

## ⚙️ Installation

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

### Backend Setup

```bash
cd backend

npm install

npm run dev
```

Backend runs on:

```bash
http://localhost:5000
```

---

## 🔐 Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret
```

Example:

```env
PORT=5000
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/moodboard
JWT_SECRET=mySuperSecretKey
```
