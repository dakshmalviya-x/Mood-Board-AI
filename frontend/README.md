## ✨ What this project does

MoodBoard AI helps users capture their emotions through a beautiful journal experience and turn reflections into:

- a visual mood snapshot
- meaningful emotional summaries
- color-based mood palettes
- personal history and insight views

It is crafted to look modern, elegant, and less generic than typical AI apps.

## 🎨 Key features

- Immersive landing experience with animated visuals
- Secure authentication and protected user flows
- Journal entry analysis with mood interpretation
- Mood history and archive browsing
- Weekly reflection insights
- Responsive and polished UI for desktop and mobile

## 🛠️ Tech stack

- React
- Tailwind CSS
- Framer Motion
- React Router
- Node.js + Express
- MongoDB
- JWT-based authentication
